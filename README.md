# Ignite Shop

## Description

This is a fullstack e-commerce application built with Next.js, Stripe, and Stitches. It allows users to browse products, add them to their cart, and checkout using Stripe.

## Features

* Browse products
* Add products to cart
* Checkout using Stripe
* Responsive design

## Prerequisites

* Node.js v16 or later
* npm v7 or later
* A Stripe account

## Installation

1. Clone the repository
2. Install dependencies with `npm install`
3. Create a `.env` file and add your Stripe API keys
4. Run the development server with `npm run dev`
5. Open http://localhost:3000 in your browser

## Usage
